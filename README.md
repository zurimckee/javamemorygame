### overview
a java-based memory matching game where users flip cards to find matching pairs. the game tracks attempts and enforces turn-based logic. it also plays a cool little song when you solve the board!

### features
- event-driven user interaction (mouse clicks)
- state management for flipped cards and matches
- board shuffling algorithm
- level picking feature
- game loop and win condition detection
